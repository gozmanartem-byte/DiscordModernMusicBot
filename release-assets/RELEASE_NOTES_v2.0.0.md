# DiscordModernMusicBot v2.0.0

## Highlights

- Full Control Panel redesign with a neon/synthwave identity and a new three-zone layout (sidebar, player center, right rail).
- Theming system with 9 built-in themes plus rounded/square corner styles (Neon Dark default).
- Integrated startup/splash animation aligned to the selected theme.
- Host dashboard upgrades: live queue preview, quick actions, queue controls (remove/shuffle/clear), and Play Next support.
- Discord player overhaul with persistent panel updates, per-second timer refresh while playing, paginated queue view, and queue action buttons.

## Control Panel

- Player controls regrouped and streamlined (playback, add/play-next, volume/bass controls with reset, EarRape host-only toggle).
- Console hidden by default with a toggle to show/hide.
- Improved macOS rendering and spacing.
- Player "Show/Remove in Discord" toggle for quick embed management.
- Enter-to-submit for song/URL input.

## Bot and Queue

- Desktop toggle for bot-only stop cleanup.
- Queue actions available from both desktop and Discord UI.

## Packaging

- Native installers and archives for macOS, Windows, Linux x86_64, and Linux aarch64.
- Linux builds include AppImage, DEB, and RPM outputs.

## Notes

- Commands remain in English by design; Control Panel and status UI are localized.
- Some Control Panel changes (theme and corner style) may require a restart to fully apply.
